1. 개요

프로그래밍 언어(Programming Language)는 컴퓨터에 명령을 전달하기 위해 사용되는 형식 언어이다. 인간이 이해할 수 있는 형태로 알고리즘을 표현하며, 컴파일러나 인터프리터를 통해 기계어로 변환되어 실행된다.

현재 전 세계적으로 수백 가지의 프로그래밍 언어가 존재하며, 각각 고유한 특성과 용도를 가지고 있다. TIOBE 인덱스 기준 2024년 6월 현재 Python, C, C++, Java, C# 순으로 인기가 높다.

2. 역사

2.1. 초기 언어

최초의 프로그래밍 언어는 1950년대에 등장했다. FORTRAN(1957)은 과학 계산용으로, COBOL(1959)은 비즈니스 데이터 처리용으로 개발되었다.

초기 프로그래밍 언어 연표 (클릭하여 펼치기)
연도 언어 개발자 특징
1957 FORTRAN IBM (John Backus) 최초의 고급 프로그래밍 언어
1958 LISP John McCarthy 함수형 프로그래밍의 시초
1959 COBOL Grace Hopper 외 비즈니스 데이터 처리
1964 BASIC Kemeny, Kurtz 교육용 프로그래밍 언어
1970 Pascal Niklaus Wirth 구조적 프로그래밍 교육용

2.2. 현대 언어

1990년대 이후 인터넷의 발전과 함께 다양한 현대 프로그래밍 언어가 등장했다. Java(1995), Python(1991), JavaScript(1995), C#(2000) 등이 대표적이다.

특히 2010년대 이후에는 Rust(2010), Go(2009), Kotlin(2011), Swift(2014), TypeScript(2012) 등 기존 언어의 단점을 보완한 새로운 언어들이 빠르게 성장하고 있다.

3. 분류

3.1. 패러다임별 분류

  • 절차적 프로그래밍: C, Pascal, FORTRAN
    • 순차적 명령 실행에 중점
    • 함수/프로시저 단위로 코드 구조화
  • 객체지향 프로그래밍: Java, C#, Python, C++
    • 객체(데이터 + 메서드) 중심 설계
    • 캡슐화, 상속, 다형성 지원
  • 함수형 프로그래밍: Haskell, Erlang, F#, Clojure
    • 불변성과 순수 함수 중심
    • 부작용 최소화
  • 논리형 프로그래밍: Prolog
    • 논리적 규칙 기반 문제 해결

4. 주요 언어

Python (클릭하여 상세 보기)

1991년 귀도 반 로섬(Guido van Rossum)이 개발한 범용 프로그래밍 언어. 간결한 문법과 풍부한 라이브러리 생태계로 데이터 과학, 웹 개발, 자동화 등 다양한 분야에서 활용된다.

# Python 예제
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

for num in fibonacci(10):
    print(num)
C# (클릭하여 상세 보기)

2000년 Microsoft가 .NET 플랫폼을 위해 개발한 객체지향 프로그래밍 언어. Anders Hejlsberg가 설계했으며, Java의 영향을 받았으나 독자적인 발전을 이루었다.

// C# 예제
public static IEnumerable<int> Fibonacci(int count)
{
    int a = 0, b = 1;
    for (int i = 0; i < count; i++)
    {
        yield return a;
        (a, b) = (b, a + b);
    }
}

foreach (var num in Fibonacci(10))
    Console.WriteLine(num);

6. 미래 전망

AI와 머신러닝의 발전으로 프로그래밍 언어의 역할도 변화하고 있다. AI 기반 코드 생성 도구의 등장으로 프로그래밍의 진입 장벽은 낮아지는 반면, 시스템 수준의 이해와 아키텍처 설계 능력의 중요성은 더욱 커질 전망이다.
