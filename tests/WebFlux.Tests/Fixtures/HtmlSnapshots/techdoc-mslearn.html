Getting started with ASP.NET Core

Article Last updated: 06/15/2024 12 min read

This tutorial teaches you the basics of building a web app with ASP.NET Core. You'll learn how to create a project, add a controller, and run your first application.

Note: This tutorial uses .NET 8. Make sure you have the latest SDK installed.

Prerequisites

  • .NET 8 SDK or later
  • Visual Studio 2022 or Visual Studio Code
  • C# Dev Kit extension (for VS Code)

Create a web app project

Open a terminal and run the following commands:

.NET CLI
Visual Studio
dotnet new webapp -o MyWebApp
cd MyWebApp

The dotnet new webapp command creates a new ASP.NET Core web app in the MyWebApp directory. The project template includes:

File/Folder Description
Program.cs App entry point and middleware configuration
Pages/ Razor Pages for the UI
wwwroot/ Static files (CSS, JS, images)
appsettings.json Configuration settings

Examine Program.cs

var builder = WebApplication.CreateBuilder(args);

// Add services to the container.
builder.Services.AddRazorPages();

var app = builder.Build();

// Configure the HTTP request pipeline.
if (!app.Environment.IsDevelopment())
{
    app.UseExceptionHandler("/Error");
    app.UseHsts();
}

app.UseHttpsRedirection();
app.UseStaticFiles();
app.UseRouting();
app.UseAuthorization();
app.MapRazorPages();

app.Run();

The preceding code:

  1. Creates a WebApplicationBuilder with preconfigured defaults
  2. Adds Razor Pages services to the dependency injection container
  3. Builds the app and configures the HTTP request pipeline
  4. Maps Razor Pages endpoints and runs the app

Run the app

dotnet run

Open a browser and navigate to https://localhost:5001. You should see the default ASP.NET Core welcome page.

Important: In production, always use HTTPS and configure proper security headers.
