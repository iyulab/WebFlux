## Expected Metadata Standards to Test:
# 1. HTML Meta Tags (title, description, keywords, author, robots)
# 2. Open Graph (og:title, og:description, og:image, og:url, og:type)
# 3. Twitter Cards (twitter:card, twitter:title, twitter:description, twitter:image)
# 4. Schema.org JSON-LD (Article, Organization, Person, SoftwareLibrary)
# 5. Dublin Core (DC.title, DC.creator, DC.subject, DC.description)
# 6. Canonical URLs (rel="canonical")
# 7. Language information (lang, hreflang)
# 8. Article metadata (article:published_time, article:author)
# 9. Site navigation (breadcrumbs, site structure)
# 10. Theme and visual metadata (theme-color, favicon)
# 11. Mobile optimization (viewport, app metadata)
# 12. Social media metadata (site_name, locale)
# 13. Publication information (last-modified, publication-date)
# 14. Category and classification metadata
# 15. Rights and licensing information
