WebFlux SDK

AI 최적화 웹 콘텐츠 처리 라이브러리

개요

WebFlux는 RAG (Retrieval-Augmented Generation) 시스템을 위한 고성능 웹 콘텐츠 크롤링 및 청킹 라이브러리입니다.

주요 특징

  • 15개 웹 메타데이터 표준 지원
  • 84% 메모리 절약 달성
  • AI 기반 지능형 청킹 전략
  • Microsoft.Playwright 동적 크롤링

기능

Web Intelligence Engine
